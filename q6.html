Difference b/w HTML & HTML 5

HTML HTML5
Limited semantic elements. Introduces semantic elements such as header, footer, nav, article, section, etc., which provide more meaningful structure to web documents.
No native storage mechanism. Introduces the localStorage and sessionStorage APIs for client-side storage, allowing web applications to store data locally in the browser.
Limited support for multimedia. Provides native support for audio and video elements, eliminating the need for third-party plugins like Flash.
